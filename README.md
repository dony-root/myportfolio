# 👨‍💻 IT Hacker Portfolio

Sebuah website portfolio pribadi interaktif dengan tema IT Hacker Terminal. Didesain untuk menampilkan proyek, keahlian, dan profil dengan gaya visual neon yang futuristik.

🔗 **[Lihat Live Website Disini](https://dony-root.github.io/myportfolio)** 

## ✨ Fitur Utama
* **IT Hacker UI:** Skema warna gelap (dark mode) dengan aksen neon hijau `#00FF88` yang khas.
* **Custom Animations:** Efek teks *glitch*, *scanlines* gaya monitor CRT jadul, kursor terminal yang berkedip, dan efek *hover glow* pada kartu proyek.
* **Fully Responsive:** Tampilan rapi dan menyesuaikan di berbagai ukuran layar (Desktop & Mobile).
* **Local Assets:** Semua *font* dan ikon dilayani secara lokal untuk performa maksimal tanpa bergantung pada CDN.

## 🛠️ Teknologi yang Digunakan
* **HTML5** (Semantik dan rapi)
* **Tailwind CSS v4** (Framework CSS modern untuk styling super cepat)
* **Custom CSS3** (Untuk animasi *keyframes* kompleks seperti efek glitch)
* **Font Awesome 6** (Ikon)
* **Google Fonts** (Space Grotesk & JetBrains Mono)

## 🚀 Cara Menjalankan di Komputer Lokal
Jika ingin mengunduh dan memodifikasi kode ini di komputermu:

1. Clone repository ini:
   git clone [https://github.com/dony-root/myportfolio.git](https://github.com/dony-root/myportfolio.git)

2. Buka folder proyek:
   cd myportfolio
    
3. Install dependencies untuk Tailwind CSS:
   npm install

4. Jalankan compiler Tailwind CSS di terminal (mode watch):
   npx @tailwindcss/cli -i ./src/style.css -o ./dist/output.css --watch
   
5. Buka file index.html di browser atau gunakan ekstensi Live Server di VS Code.
